# Gestión de Citas Inteligente

## Reglas del Repositorio

Este proyecto trabaja con GitFlow.
El incumplimiento de estas reglas puede generar conflictos en el código.

---

## ❌ Prohibido

- Hacer push directo a `main`
- Trabajar directamente en `develop`
- Hacer merge sin Pull Request
- Borrar archivos de otros compañeros
- Subir código que no compila

---

## ✅ Forma correcta de trabajar

### 1. Descargar proyecto
```bash
git clone https://github.com/sparyock/Gestion-De-Citas-Inteligente.git
cd Gestion-De-Citas-Inteligente
git checkout develop
git pull origin develop
